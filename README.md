# Tetris-Game
# 🎮 Tetris Game

A classic **Tetris Game** built using **HTML, CSS, and JavaScript**. Arrange the falling blocks to complete full rows, clear lines, and score as many points as possible.


## ✨ Features

- Classic Tetris gameplay
- Responsive keyboard controls
- Random tetromino generation
- Score tracking
- Line clearing mechanics
- Game Over detection
- Fully browser-based

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬅️ Left Arrow | Move Left |
| ➡️ Right Arrow | Move Right |
| ⬇️ Down Arrow | Move Down |
| ⬆️ Up Arrow | Rotate Block |

---

## 💻 Run Locally

1. Clone the repository:

```bash
git clone https://github.com/usaidkhot159/Tetris-Game.git
```

2. Open the project folder.

3. Open `index.html` in your browser or run it using the **Live Server** extension in VS Code.
